# Skill: ready-check

## Description
Performs a quick internal audit to confirm the assistant is calibrated and ready for new commands. This skill is a formalization of the user's repeated instruction: "Review the workspace instructions, refresh your sense of the user's preferences, and quietly prepare for the next conversation turn."

## Actions
1. Read `SOUL.md` to refresh persona.
2. Read `AGENTS.md` to refresh operating instructions.
3. Read `USER.md` to refresh user context.
4. Output a concise confirmation message, such as "Ready."
